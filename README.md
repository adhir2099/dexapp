# Set up
<p>Clone the repository</p>

```bash
git clone https://github.com/adhir2099/poke-dex-app.git
```
**Running the Project**
<p>Head to it</p>
<p>Open a terminal and run the following commands.</p>

```bash
npm i
npm run dev
```
